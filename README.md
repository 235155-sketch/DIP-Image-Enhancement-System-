# Smart Image Enhancement System
This repository contains a complete Image Processing pipeline developed in MATLAB.

## Project Overview
- **Task 6.1-6.2:** Image Acquisition, Sampling, and Quantization.
- **Task 6.3:** Geometric Transformations (Rotation, Translation, Shearing).
- **Task 6.4:** Intensity Transformations (Log, Gamma, Negative).
- **Task 6.5-6.6:** Histogram Processing and Final System Integration.

## How to Use
1. Clone the repository.
2. Run `code` in MATLAB.
3. Check the `images/output/` folder for results.
